Description

This module together with onviso_server is the foundation of a configuration back-end layer for higher-level tools which want to make use of the Onviso. The onviso_server acts as a middle man between any user-interface and the wrapper, and offers a possibility to store and load configurations to and from files.

For a full example of how these two modules can be used, please refer to the onviso_server module.

Version: 1.0
